# Manifesto UAP OS

## Missão

O UAP OS (Universal Automation Platform) é uma plataforma universal para criação de aplicações de automação sem necessidade de programação.

O objetivo é permitir que qualquer pessoa desenvolva sistemas de automação utilizando apenas componentes visuais e configurações, sem escrever código.

---

## Visão

Ser a principal plataforma mundial para automação universal baseada em componentes reutilizáveis.

---

## Princípios

* Nenhuma linha de programação será necessária para o usuário final.
* Todo equipamento será tratado como um componente.
* Toda automação será construída visualmente.
* Todo projeto poderá ser reutilizado.
* Toda configuração será armazenada em formato aberto.
* O sistema deverá funcionar inicialmente em Raspberry Pi, podendo futuramente operar em outros equipamentos.

---

## O que é um Projeto

Um projeto representa uma automação completa.

Exemplos:

* Lavanderia
* Vending Machine
* Restaurante
* Automação Residencial
* Controle de Acesso
* Irrigação
* Indústria

---

## O que é um Componente

Um componente representa qualquer dispositivo físico ou lógico.

Exemplos:

* Relé
* Sensor
* Motor
* Display
* Impressora
* Botão
* Timer
* Variável
* MQTT
* HTTP
* PIX

---

## O que é um Fluxo

Um fluxo representa a ligação entre componentes.

Os fluxos definem a lógica da automação.

---

## Objetivo Principal

Transformar ideias em automações funcionais através de uma interface visual simples, rápida e universal.

O UAP OS deve permitir que uma mesma plataforma controle desde pequenos projetos domésticos até aplicações comerciais e industriais.
